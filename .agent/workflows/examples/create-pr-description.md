### Purpose of this PR

This pull request implements the **frontend logic for displaying UPS battery status** across both **Before-Surgery** and **In-Surgery** pages. Backend integration will be handled in a future update.

---

### Summary of Changes (across both PRs)

- Added **battery status display** to the UI on both **Before-Surgery** and **In-Surgery** screens
- Displayed real-time **charging status** and **battery level**
- Shown **warning with estimated remaining time** when battery is low
- Displayed **error/warning message** when the UPS device is **unable to charge**

> Note: This PR focuses on UI implementation only — no backend APIs are connected yet.
